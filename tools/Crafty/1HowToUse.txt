Garrys mod:
1.Extract workshop package with gmad
2.Decompile map with BSPSOURCE including files (Extract embedded files)
3.Rename folder with extracted files to vbsp.zip
4.Move other files from package inside this folder
5.Open vmf map.
6.Go to file system and mount vbsp.zip folder.
7.Reopen file
